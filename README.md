# csvtoascii

Golang CSV converter to ASCII table

# How to use it

Clone the repo

```
git clone https://github.com/ImValerio/csvtoascii.git
```

Compile the source code

```
cd csvtoascii
go build .
```

Execute it

```
./csvtoascii file.csv
```

To convert multiple files, specify their names as arguments.

```
./csvtoascii "file.csv;file2.csv"
```
